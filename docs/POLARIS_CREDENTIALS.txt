===================================

These credentials are randomly generated when Polaris starts.
To find the current credentials, run:

  docker logs demo-data-platform-query-polaris-1 2>&1 | grep "root principal credentials:"

OAuth Token Endpoint:
  POST http://localhost:8181/api/catalog/v1/oauth/tokens

To obtain an access token:
  curl -X POST http://localhost:8181/api/catalog/v1/oauth/tokens \
    -H "Content-Type: application/x-www-form-urlencoded" \
    -d "grant_type=client_credentials" \
    -d "client_id=<YOUR_CLIENT_ID>" \
    -d "client_secret=<YOUR_CLIENT_SECRET>" \
    -d "scope=PRINCIPAL_ROLE:ALL"

Important Notes:
- Polaris uses H2 in-memory database by default (configured in docker-compose.yml)
- Credentials are generated fresh on each startup
- To use PostgreSQL, configure QUARKUS_DATASOURCE_JDBC_URL accordingly
- The scope "PRINCIPAL_ROLE:ALL" is required for OAuth token requests
